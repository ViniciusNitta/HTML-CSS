Imagens Dinâmicas

Usando a tag PICTURE podemos definir para uma mesma imagens varías mídias de tamanhos diferentes para se adaptar ao tamanho de tela, devemos usar a tag PICTURE e dentro dela inserir as tags IMG e SOURCE onde IMG é a imagem padrão e SOURCE são as imagens secundárias que vão ser exibidas quando a resolução da tela do usuário tiver resoluções diferentes.

**Tomar cuidado com a ordem dentro de PICTURES, as ordens devem estar em sequência descrescente ou crescente.

Áudios

Áudios podem ser adicionados usando a tag AUDIO e preenchendo o SRC com o arquivo que deve ser tocado, porém está maneira não é muito indicada pois o navegador pode não ter compatibilidade com o arquivo da música.

** Alguns formatos podem ser usados além do MP3 seriam estes o WAV e OGG.

Para integrar áudios dentro do site usamos a tag AUDIO, colocamos alguns SOURCEs que vão conter os áudios em formatos diferentes e iremos colocar dentro os atributos, sendo alguns destes o CONTROLS, AUTOPLAY, LOOP e PRELOAD.

  • controls: irá exibir um player na tela para poder pausar ou reproduzir a música;
  • autoplay: irá tocar a música assim que o site for carregado;
  • loop: irá colocar em loop a música;
  • preload: irá definir o que deverá acontecer com o áudio assim que o site for carregado, tendo 3 varíaveis;
    • metadata: vai carregar as informações do arquivo, mas não o arquivo em si;
    • none: não irá carregar nada, nem informações nem o arquivo, so irá carregar assim que o usuário clicar para reproduzir o áudio;
    • auto: sendo este o padrão, irá carregar o arquivo de áudio mesmo que o mesmo nunca seja reproduzido.

Vídeos Hospedados no Próprio Servidor

Vídeos são inseridos usando a tag VIDEO e usam os mesmos atributos que o AUDIO (controls, autoplay, loop e predata), com o diferencial de ter a opção WIDTH e THUMB onde deve ser pré-definido o tamanho que o vídeo será exibido e a imagem que vai aparecer antes da exibição.

** Uma dica para converter arquivos é o programa Handbrake que é open source
