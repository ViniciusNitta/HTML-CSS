Código-fonte e Pré Formatação

Em algumas situações precisamos destacar determinados termos de maneira que tornem-se facilmente identíficaveis, como por exemplo códigos-fonte, para isso usamos a tag CODE que irá dar um efeito visual único de monoespaçamento.
"O comando document.getElementById('test') é escrito em JavaScrip.""

Usando a tag CODE conseguimos o monoespaçamento, porém isso não organiza as linhas, para isso podemos usar a tag PRE que irá apresentar os termos como eles foram pré-estabelecidos, inclusive espaços e linhas vazias.

Apenas com a tag CODE:

num = input(Digite um número inteiro) if num % 2 == 0: print(f'O número {num} é par.') else: print(f'O número {num} é ímpar.') print('Fim do programa')

Com a tag PRE :

                
                            num = input(Digite um número inteiro)
                            if num % 2 == 0:
                                print(f'O número {num} é par.')
                            else:
                                print(f'O número {num} é ímpar.')
                            print('Fim do programa')

**Selecionar as linhas e apertar Shift+Tab recua todo o conteúdo sem alterar a formatação,caso queira avançar o conteúdo é só selecionar as linhas e apertar Tab

Citações Simples

Em citações utilizamos a tag Q que adiciona as aspas.

Como diria o pai de um amigo: O computador é um burro muito rápido

Citações Completas

Em citações mais completas podemos usar a tag BLOCKQUOTE que irá modificar um pouco a formatação do texto:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Molestiae accusantium expedita ex sed cum itaque, facere atque facilis consectetur velit veniam quae aliquid dolor doloremque iure esse ab eligendi exercitationem?
