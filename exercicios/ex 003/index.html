Adicionando imagens em HTML

É possível adicionar imagens em sites digitando 'img' e selecionando a primeira tag:
Em 'src=' selecionar a imagem.
E em 'alt=' adicionar uma descrição para a imagem.
**Clicando no meio das aspas de 'src' e apertando Ctrl+Space é possível adicionar a imagem diretamente, contanto que ela esteja na mesma pasta que o arquivo HTML.

Também é possível adicionar imagens que estão dentro de outras pastas, contanto que essa pasta esteja dentro da pasta do HTML.

É possível também adicionar imagens que estão dentro da internet, para isso basta adicionar a URL inteira da imagem dentro de 'src'.
