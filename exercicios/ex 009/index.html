Listas

Listas ordenadas

É possível criar listas ordenadas usando a tag OL(ordered lists) e completando com a tag LI(list itens), exemplo de lista:

  1. Amendoim
  2. Sem
  3. Casca

A tag OL pode ser alterada usando o TYPE para algumas variações no lugar do número(está sendo a configuração padrão):

  1. Amendoim
  2. Sem
  3. Casca
  1. Amendoim
  2. Sem
  3. Casca
  1. Amendoim
  2. Sem
  3. Casca
  1. Amendoim
  2. Sem
  3. Casca

Em sequência temos o uso das tags:
<ol>, modelo padrão, utiiza números;
<ol type="A">, altera para letra maiúscula;
<ol type="a">, altera para letra minúscula;
<ol type="I">, altera para algarismos romanos maiúsculos;
<ol type="i">, altera para algarismos romanos minúsculos.

Listas não ordenadas

As listas não ordenadas são listas que não possuem uma sequência a ser seguida, dessa maneira elas apresentam apenas os pontos, mas não os classifica, estás podem ser criadas usando a tag UL(unordered list) e colocando as LI dentro delas normalmente.

  • Amendoim
  • Sem
  • Casca

Existem também algumas variações dentro da tag UL usando o complemento TYPE:

  • Amendoim
  • Sem
  • Casca
  • Amendoim
  • Sem
  • Casca

Em sequência temos o uso das tags:
<ul>, modelo padrão, utiliza o círculo preenchido;
<ul type="circle">, altera para um círculo vazio;
<ul type="square">, altera para um quadrado;

Misturando as listas

É possível criar listas dentro de listas, listas ordenadas com listas não ordenadas, com isso abre muitas possibilidades de modelos diferentes, como por exemplo:

**Caso seja necessário começar a lista fora de ordem, por exemplo começar pelo número 3,4,5... , é só adicionar start="x" depois da tag OL ou UL, e caso tenha TYPE, depois do type.

Lista de definições

É possível criar listas onde não existem marcadores, está lista é como se fosse um dicionário, onde tem o termo e depois a sua definição, para criar este tipo de lista utiliza-se a tag DL(definition list) e dentro dessa lista nós temos as DT(definition terms) e as DD(definition description), por exemplo:

HTML
Linguagem de marcação onde está inserido o conteúdo do site, trabalha com tags semânticas onde definimos significados ao conteúdo.
CSS
Linguagem de marcação para a parte de estilo, está trabalha com a parte visual do site, nela definimos principalmente a forma do conteúdo.
HTML
Linguagem de programação para as interações do site, nela trabalhamos com códigos que visam uma melhor experiência ao usuário por meio de interações.

** Uma dica para mudar varías tags iguais de uma vez é segurar Alt e ir selecionando as tags que devem ser mudadas com o cursor do mouse no meio delas.
