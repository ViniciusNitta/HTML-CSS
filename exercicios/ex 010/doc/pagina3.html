Terceira página

Aqui temos nossa terceira página, está página não está na mesma pasta que o index e a página2, por isso para acessarmos outros arquivos devemos usar o complemento ../ este irá fazer com que seja procurado os documentos necessários na pasta anterior a que se encontra este arquivo, por exemplo para voltarmos ao index vamos ter que usar a tag <a href="../index.html"> em vez de usar a tag <a href="index.html"
