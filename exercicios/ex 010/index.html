Criando links dentro do site

HyperLink/Âncoras

A criação de links dentro de sites melhora a experiência do usuário dentro de um site, deixando mais fácil que ele encontre conteúdos relacionados com o que ele busca encontrar, usando a tag A e preenchendo o atributo href podemos criar um hyperlink que redireciona o usuário para o link preechido.

  • Acesse o link para ter aulas de programação gratuitas!
  • Acesse o link para ir até o YouTube!

Outro atributo importante é o hreflang onde colocamos qual é a língua do site que está no hyperlink, assim quando o usuário acessar o link ele terá a opção de tradução da pagína.

Site da W3Schools!

Um problema encontrado nessas situações é que o hyperlink redireciona o usuário diretamente para o link, que pode acabar perdendo a pagína atual, o método de resolver isso é usando a tag TARGET, que pode ser configurada de duas maneiras, _self e _blank
_self irá abrir o link na pagína atual, sendo este a configuração padrão.
_blank irá abrir o link em outra página.

  • Site da W3Schools!(Ir para o site)
  • Site da W3Schools!(Abrir o site em uma nova guia)

Outra funcionalidade interessante é indicar qual é a natureza de destino, para isso usamos a tag REL, está aceita varías funções, entre elas:

  1. next: indica que o link é a continuação do link atual;
  2. prev: indica que o link é a parte anterior do link atual;
  3. author: indica que é um link para o autor do site;
  4. exthernal: indica que o link é para um site externo;
  5. nofollow: indica que o link é para um site não endossado, como um site pago.

Este é um exemplo de um link interno onde o usuário vai ser redirecionado para uma outra pagína dentro do mesmo site, dessa maneira não é necessário ter a URL completa.

  • Continuar navegando na segunda página
  • Ir para nossa terceira página

Já este é um link externo onde o usuário vai ser redirecionado para uma outra pagína em outro site, dessa maneira sendo necessário a URL inteira inclusive os protocolos https:// e http://

Downloads

Para criar downloads devemos criar um hyperlink(âncora) e adicionar a funcionalidade download dentro da tag, outro ponto importante é colocar o tipo de arquivo que está sendo disponibilizado, para isso deve ser usado o type, por exemplo:
